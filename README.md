# Lost-and-Found-Portal
The Lost and Found Portal is a web application built using HTML, CSS, and JavaScript. It helps users report lost or found items with location details. The portal displays items on an embedded Google Map, making it easy to visually track where objects were last seen.
## How It Works

1. User chooses **Lost** or **Found** category  
2. Fills item details in the form  
3. Selects the location on Google Maps  
4. Submits the report  
5. Item appears in the list + map marker  
6. Other users can browse items to identify matches

- Students building portfolio projects  
- Beginners learning Google Maps API  
- Resume-ready web development use  
- Demonstrating HTML, CSS, and JavaScript skills
